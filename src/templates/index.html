Данная страничка поможет вам перемножить строки вашей книги Excel.

Например, конвертировать два ваших листа:

Столбец A Столбец B
AAA 111
BBB 222
CCC 333
в =>
Столбец A Столбец B
AAA 111
AAA 222
AAA 333
BBB 111
BBB 222
BBB 333
CCC 111
CCC 222
CCC 333

Пожалуйста, выберете одну книгу Excel которая отвечает следующим требованиям:

  1. Расширение файла ".xlsx".
  2. Имеет более одного листа.
  3. Можно использовать сколько угодно листов, они все перемножаются.
  4. Если в результирующем массиве строк будет более 1 миллиона, то книга на выходе будет с несколькими листами.
  5. В результирующем массиве строк должно быть менее 3 миллионов.
